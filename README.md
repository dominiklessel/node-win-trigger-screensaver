# node-win-start-screensaver

> Starts the default screensaver

## Install

```
$ npm install --save node-win-start-screensaver
```

## Usage

```
startScreensaver((err) => {
  if (!!err) {
    throw err;
  }
});
```

## API

### startScreensaver(callback)

#### callback

*Required*  
Type: `function`
